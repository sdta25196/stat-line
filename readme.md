# A line of code statistics tool and can assign file type

### install

```js
  npm i stat-line -g
```

### use

```js
  line path [--type]
```

### e.g
```js
  // current path default statistics .js file
  line   
```
  or 
```js
  // statistics current path .scss file
  line ./ --type .scss 
```
