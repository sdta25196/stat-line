

Usage: line [path] [Options]

path                If you want to use [path]. You need to use relative path.
                    default [path] is [./] 
                    e.g line ./src

Options:

  -t, --type        specify the type of file to be counted, multiple types can be specified
                    e.g  line --type scss
                    e.g  line --type scss css js   // will count scss、css、js file

  -h, --help        Display help information
                    e.g  line --help
