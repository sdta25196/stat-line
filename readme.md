# A line of code statistics tool and can assign file type

[中文文档](https://github.com/sdta25196/stat-line/blob/master/readme_CN.md)

### Installation

```js
  npm i stat-line -g
```

### Usage

```js
  line [path] [--type]
```

### Examples
```js
  // current path default statistics .js file
  line   
```
  or 
```js
  // statistics current path .scss file
  line ./ --type .scss 
```
